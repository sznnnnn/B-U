# 个人档案卡片页 - 设计说明

> Notion 风格的个人档案页面  
> 完全参考 Notion 的设计语言

---

## 🎨 设计特点

### Notion 风格特征

1. **极简设计**
   - 大量留白（卡片间距 20px）
   - 简洁的排版
   - 清晰的层次结构

2. **卡片式布局**
   - 每个信息模块独立成卡片
   - 柔和的边框（1px #e9e9e7）
   - 圆角设计（6px）
   - 悬停效果（轻微阴影）

3. **配色方案**
   - 背景：`#ffffff`（纯白）
   - 文字主色：`#37352f`（深灰）
   - 文字次色：`#787774`（中灰）
   - 边框：`#e9e9e7`（浅灰）
   - 强调色：`#ff6b35`（BuddyUP 橙色）

4. **字体系统**
   - 主字体：`-apple-system, BlinkMacSystemFont, "Segoe UI"`
   - 中文字体：`"Noto Sans SC"`
   - 标题：700 粗体
   - 正文：400 常规
   - 辅助文字：500 中等

5. **交互细节**
   - 卡片悬停效果
   - 淡入动画
   - 进度条动画
   - 平滑过渡

## 🎯 卡片类型

### 1. 个人资料卡片（大卡片）

**特点：**
- 横跨整行
- 包含头像、姓名、角色、标签
- 头像使用渐变背景（BuddyUP 橙色）

**内容：**
- 头像（圆形，80x80px）
- 姓名（28px，粗体）
- 角色（16px，中灰）
- 标签（多个，圆角背景）

---

### 2. 进度卡片

**特点：**
- 显示申请进度百分比
- 进度条动画效果
- 数字突出显示

**内容：**
- 进度百分比（24px，粗体）
- 进度描述（14px，中灰）
- 进度条（6px 高度，渐变填充）

---

### 3. 列表卡片

**特点：**
- 图标 + 标签 + 值
- 列表项之间有分隔线
- 图标使用 emoji

**内容：**
- 图标（20x20px）
- 标签（14px，中灰）
- 值（16px，深灰，中等粗细）

---

### 4. 经历卡片

**特点：**
- 多个经历项
- 每项包含标题、时间、描述
- 项之间有分隔线

**内容：**
- 经历标题（16px，粗体）
- 经历元信息（13px，中灰）
- 经历描述（14px，深灰）

---

### 5. 文本卡片

**特点：**
- 大段文本内容
- 行距舒适（1.7）
- 支持换行

**内容：**
- 文本内容（15px，深灰）
- 支持加粗、强调等格式

## 🎨 设计规范


## 💻 技术实现

### HTML 结构

```html
<div class="container">
    <div class="page-header">
        <h1 class="page-title">个人档案</h1>
        <p class="page-subtitle">你的申请信息与进度概览</p>
    </div>
    
    <div class="cards-grid">
        <div class="card">...</div>
        <div class="card">...</div>
        ...
    </div>
</div>
```

### CSS 关键点

1. **Grid 布局**
   ```css
   .cards-grid {
       display: grid;
       grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
       gap: 20px;
   }
   ```

2. **卡片悬停效果**
   ```css
   .card:hover {
       box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
       border-color: #d4d4d1;
   }
   ```

3. **淡入动画**
   ```css
   @keyframes fadeIn {
       from {
           opacity: 0;
           transform: translateY(10px);
       }
       to {
           opacity: 1;
           transform: translateY(0);
       }
   }
   ```

### JavaScript 交互

```javascript
// 卡片点击事件
document.querySelectorAll('.card').forEach(card => {
    card.addEventListener('click', function() {
        // 交互逻辑
    });
});

// 进度条动画
window.addEventListener('load', function() {
    const progressFill = document.querySelector('.progress-fill');
    if (progressFill) {
        setTimeout(() => {
            progressFill.style.transition = 'width 0.8s ease';
        }, 100);
    }
});
```

- **桌面端**：> 768px（2 列布局）

### 移动端调整

