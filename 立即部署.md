# 🚀 立即部署到公网 - 让别人也能访问

## ⭐ 最简单方式：Netlify 拖拽部署（推荐）

### 3 步完成部署：

1. **打开 Netlify**
   - 访问：https://www.netlify.com/
   - 点击右上角 "Sign up" 注册（可用 GitHub 账号登录）

2. **拖拽文件夹**
   - 登录后，在首页找到 "Add new site" > "Deploy manually"
   - 将 `03-设计原型` 文件夹直接拖拽到部署区域
   - 等待几秒钟

3. **获得链接**
   - 部署完成后，你会得到一个链接，例如：
   - `https://amazing-name-123456.netlify.app`
   - 这个链接可以分享给任何人访问！

---

## 或者使用 Vercel（需要登录）

### 步骤：

1. **登录 Vercel**
   ```bash
   cd /Users/szn/Downloads/文书/03-设计原型
   vercel login
   ```

2. **部署**
   ```bash
   vercel --prod
   ```

3. **获得链接**
   - 部署完成后会显示访问链接
