# 部署指南

本指南提供了多种部署静态HTML文件的方式。

## 文件说明

- `03-设计原型/个人简介-Notion风格.html` - 个人简介页面
- `03-设计原型/个人文书生成-Notion风格.html` - 个人文书生成页面

## 部署方式

### 方式一：GitHub Pages（推荐）

1. **创建 GitHub 仓库**
   ```bash
   cd /Users/szn/Downloads/文书
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push -u origin main
   ```

2. **启用 GitHub Pages**
   - 进入仓库 Settings
   - 找到 Pages 选项
   - Source 选择 `main` 分支
   - Folder 选择 `/ (root)` 或 `/docs`
   - 保存后访问：`https://你的用户名.github.io/你的仓库名/03-设计原型/个人简介-Notion风格.html`

### 方式二：Netlify（最简单）

1. **访问 [Netlify](https://www.netlify.com/)**
2. **拖拽部署**
   - 登录 Netlify
   - 将整个 `03-设计原型` 文件夹拖拽到 Netlify 部署区域
   - 自动生成访问链接

3. **命令行部署**
   ```bash
   npm install -g netlify-cli
   cd /Users/szn/Downloads/文书/03-设计原型
   netlify deploy
   netlify deploy --prod
   ```

### 方式三：Vercel

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **部署**
   ```bash
   cd /Users/szn/Downloads/文书/03-设计原型
   vercel
   ```

### 方式四：本地服务器（开发测试）

使用 Python 简单服务器：

```bash
cd /Users/szn/Downloads/文书/03-设计原型
python3 -m http.server 8000
```

然后访问：`http://localhost:8000/个人简介-Notion风格.html`

或使用 Node.js：

```bash
npx http-server 03-设计原型 -p 8000
```

## 快速部署脚本

创建一个简单的部署脚本：

```bash
#!/bin/bash
# deploy.sh

echo "开始部署..."

# 方式1: 使用 Python 服务器
cd "$(dirname "$0")/03-设计原型"
python3 -m http.server 8000

# 方式2: 或使用 Node.js
# npx http-server . -p 8000
```

## 注意事项

1. **相对路径**：所有文件使用相对路径，可以直接部署
2. **CDN 资源**：使用了 Tailwind CSS CDN，需要网络连接
3. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）

## 推荐部署流程

1. **开发测试**：使用本地服务器测试
2. **版本控制**：使用 Git 管理代码
3. **生产部署**：使用 Netlify 或 Vercel 自动部署
