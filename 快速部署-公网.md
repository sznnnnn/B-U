# 快速部署到公网 - 让别人也能访问

## 方式一：Netlify 拖拽部署（最简单，推荐）⭐

### 步骤：

1. **访问 Netlify**
   - 打开浏览器访问：https://www.netlify.com/
   - 点击 "Sign up" 注册账号（可以使用 GitHub 账号登录）

2. **拖拽部署**
   - 登录后，在 Netlify 首页找到 "Add new site" > "Deploy manually"
   - 将 `03-设计原型` 文件夹直接拖拽到部署区域
   - 等待几秒钟，自动生成访问链接

3. **获取访问链接**
   - 部署完成后，你会得到一个类似这样的链接：
   - `https://random-name-123456.netlify.app`
   - 这个链接可以分享给任何人访问

4. **自定义域名（可选）**
   - 在 Netlify 项目设置中可以自定义域名
   - 例如：`https://yourname.netlify.app`

## 方式二：Vercel 部署（也很简单）

### 步骤：

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **部署**
   ```bash
   cd /Users/szn/Downloads/文书/03-设计原型
   vercel
   ```

3. **按提示操作**
   - 首次使用需要登录（会打开浏览器）
   - 选择项目设置
   - 自动生成访问链接

## 方式三：GitHub Pages（需要 GitHub 账号）

### 步骤：

1. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 创建新仓库（例如：`personal-profile`）

2. **上传文件**
   ```bash
   cd /Users/szn/Downloads/文书
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 进入仓库 Settings
   - 找到 Pages 选项
   - Source 选择 `main` 分支
   - Folder 选择 `/ (root)`
   - 保存后访问：`https://你的用户名.github.io/你的仓库名/03-设计原型/index.html`
